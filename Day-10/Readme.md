# 📘 Day 10 – Python Modules & Mini Project

## 🚀 What I Learned Today

Today I explored Python Modules, which help organize code into reusable and structured components. This is a key concept for writing scalable and clean programs.

---

## 📂 Topics Covered

### 1. Creating a Module
- Learned how to create a custom Python module using `.py` files.
- Understood how to import and use functions from another file.

Example:
import mymodule
mymodule.my_function()

---

### 2. Variables in Module
- Accessed variables defined inside a module.
- Understood module-level scope.

Example:
import mymodule
print(mymodule.variable_name)

---

### 3. Naming a Module
- Learned best practices for naming modules:
  - Use lowercase letters
  - Avoid spaces (use `_` instead)
  - Keep names meaningful

---

### 4. Built-in Modules
- Explored Python’s built-in modules that come pre-installed.
- No need to install them separately.

---

### 5. os Module
- Used for interacting with the operating system.

Example:
import os
print(os.getcwd())

---

### 6. sys Module
- Provides access to system-specific parameters.

Example:
import sys
print(sys.version)

---

### 7. statistics Module
- Used for mathematical statistics.

Example:
import statistics
print(statistics.mean([1, 2, 3, 4]))

---

### 8. math Module
- Provides mathematical functions.

Example:
import math
print(math.sqrt(16))

---

### 9. string Module
- Contains useful string constants and helper functions.

---

### 10. random Module
- Used to generate random values.

Example:
import random
print(random.randint(1, 10))

---

## 🧠 Mini Project
- Applied all the concepts learned today.
- Built a small project using modules and functions.
- Strengthened understanding of importing and structuring code.

---

## 🔑 Key Takeaways
- Modules help in code reusability.
- Built-in modules save time and effort.
- Proper structure makes code easier to maintain.
- Python provides powerful tools through standard libraries.
