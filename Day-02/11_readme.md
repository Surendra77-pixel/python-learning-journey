## 🧠 What I Learned

### 1️⃣ Conditional Statements

Used to make decisions in Python programs.

---

### 2️⃣ `if` Statement

Executes code only if the condition is **True**.

---

### 3️⃣ `elif` Statement

Checks multiple conditions.

---

### 4️⃣ `else` Statement

Runs when all above conditions are False.

---

### 5️⃣ Shorthand If-Else (Ternary Operator)

Short and clean way to write conditions.

---

### 6️⃣ Combining Multiple Conditions

Using logical operators:

* `and`
* `or`
* `not`

---

### 7️⃣ Nested If Statements

Condition inside another condition.

---

### 8️⃣ `pass` Statement

Acts as a placeholder (does nothing).

---

### 9️⃣ `match-case` Statement (Python 3.10+)

Used for pattern matching (like switch-case).

---

### 🔟 Match Case with Advanced Concepts

#### ✔ Logical Operators

Used to combine conditions inside match-case.

#### ✔ Membership Operators

Used to check values inside lists or collections.

---

## 💡 Key Concepts Summary

* `if` → checks condition
* `elif` → multiple conditions
* `else` → default case
* `nested if` → multiple layers of checking
* `shorthand if` → compact syntax
* `match-case` → clean pattern matching
* `pass` → placeholder
* logical operators → combine conditions
* membership operators → check in lists
