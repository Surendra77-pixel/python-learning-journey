01
Variables
Variables store data in computer memory. Mnemonic variables are recommended — a mnemonic variable is a variable name that can be easily remembered and associated. A variable refers to a memory address in which data is stored.

A variable has a name to identify it, its value can change anytime, and it is created the moment you assign a value to it. It can hold different types of data — numbers, strings, lists, and so on.

The Variable Rules

Not everything is allowed when naming a variable. Numbers at the beginning, special characters, and hyphens are not allowed.

  • ✓ OKMYVAR = "JOHN" — uppercase is fine
  • ✓ OKmyvar = "JOHN" — lowercase is fine
  • ✗ NOmy-var = "JOHN" — hyphens are not allowed
  • ✗ NOmy var = "JOHN" — spaces are not allowed

Multiple Word Variable Names

When a variable name has multiple words, there are three popular conventions to follow:

Camel Case
myVariableName
Pascal Case
MyVariableName
Snake Case
my_variable_name

Assigning Multiple Values to Multiple Variables

python
x, y, z = 10, 20, 30
print(x)  # Output: 10
print(y)  # Output: 20
print(z)  # Output: 30

One Value to Multiple Variables

python
x = y = z = 10
print(x)  # Output: 10
print(y)  # Output: 10
print(z)  # Output: 10

Unpacking a Collection

If you have a collection of values in a list, tuple, or any other iterable, you can unpack them into individual variables — this is called unpacking a collection.

python
fruits = ["apple", "banana", "cherry"]
x, y, z = fruits
print(x)  # Output: apple
print(y)  # Output: banana
print(z)  # Output: cherry

Output Variables

python
x = "python"
y = "is"
z = "awesome"
print(x, y, z)   # Output: python is awesome
print(x + y + z)  # Output: pythonisawesome

Data Types
Data types define the type of value a variable can store. They tell Python what kind of data you are working with. Python has several built-in data types.
int
Integer numbers — whole numbers like 5, 100, -20
float
Floating-point numbers — decimals like 3.14, 2.5
complex
Complex numbers like 2+3j
str
Text strings — "Hello, World!"
bool
Boolean — True or False
list
Ordered, mutable collection — [1, 2, 3]
tuple
Ordered, immutable collection — (1, 2, 3)
dict
Key-value pairs — {"name": "Alice"}
set
Collection of unique values — {1, 2, 3}
None
Absence of a value — NoneType
python
x = 5             # int
y = "Hello"       # str
z = 3.14          # float
w = 2 + 3j        # complex
is_true = True    # bool
my_list = [1, 2, 3, 4, 5]             # list
my_tuple = (1, 2, 3, 4, 5)           # tuple
my_dict = {"name": "Alice", "age": 30}  # dict
my_set = {1, 2, 3, 4, 5}             # set
my_none = None                        # NoneType

Checking Data Type with type()

We can use the built-in type() function to check the data type of any variable.

python
x = "Surendra"
print(type(x))  # Output: <class 'str'>

Random Number

Python does not have a built-in random() function, but it has a random module that contains functions to generate random numbers.

python
import random
print(random.randrange(1, 10))  # Output: a random number between 1 and 10

03
Conversions
There are a few types of conversion in Python: Type Conversion, Type Casting, and Type Checking.

Type Conversion (Implicit)

The automatic conversion of one data type to another is called type conversion, also known as implicit type conversion or coercion. Python automatically converts one data type to another when it is necessary to perform an operation.

python
x = 5      # integer
y = 2.0    # float
z = x + y  # the integer 5 is automatically converted to float 5.0
print(z)   # Output: 7.0

Type Casting (Explicit)

Type casting is the process of converting a variable from one data type to another explicitly. In Python, we use built-in functions like int(), float(), str(), etc.

python
x = 5
y = float(x)  # explicitly convert integer to float
print(y)      # Output: 5.0

Type Checking

Type checking is the process of verifying the data type of a variable or expression using the type() function.

python
x = ["Apple", "Banana", "Cherry"]  # list
print(type(x))  # Output: <class 'list'>

04
Input & Output
Input is data a program receives from the user or another source. Output is data a program produces as a result of processing. In Python we use input() to get input and print() to display output.

Getting Input from the User

python
name = input("What is your name? ")
print("Hello, " + name + "!")  # Output: Hello, [name]!

age = input("What is your age? ")
print("You are " + age + " years old.")  # Output: You are [age] years old.

Outputting Data to the Console

python
x = 5
y = 10
print("The value of x is:", x)  # Output: The value of x is: 5
print("The value of y is:", y)  # Output: The value of y is: 10

05
Strings
Strings play a crucial role in Python — they represent and manipulate text data. A string is a sequence of characters enclosed in single quotes (' ') or double quotes (" "). Strings are immutable, meaning once created, they cannot be changed. You can, however, create new strings by concatenating or slicing existing ones.

Multi-line Strings

Using triple quotes allows us to create a multi-line string that spans multiple lines without escape characters.

python
print("""This is surendra
      im here to learn python""")

Strings as an Array (Indexing)

A string in Python behaves like an array of characters. Each character has an index (position), starting from 0.

python
a = "The python is the interesting language"
print(a[0])  # Output: T
print(a[1])  # Output: h

Looping Through a String

Using a for loop, we can iterate through each character in a string one by one.

python
for x in "This is surendra":
    print(x)  # prints each character on a new line

Checking the Length with len()

We use the built-in len() function to check the length of a string. It counts every character including spaces and punctuation.

python
a = "The python is the interesting language"
print(len(a))  # Output: 39

Checking if a Substring Exists

We can use in, not in, if, and if not to check whether a substring is inside a string.

python
a = "The python is the interesting language"

# Using 'in'
print("python" in a)    # Output: True

# Using 'not in'
print("java" not in a)  # Output: True

# Using 'if'
if "python" in a:
    print("Yes, 'python' is present in the string.")

# Using 'if not'
if "java" not in a:
    print("No, 'java' is not present in the string.")

Slicing Strings

We use slicing to extract a portion of a string. The syntax is string[start:end] — start is inclusive, end is exclusive.

python
b = "The python is the interesting language"
print(b[2:5])   # Output: e p  (index 2 to 4)

# Slice from start
A = "This is surendra"
print(A[:4])    # Output: This

# Slice to end
a = "The himalayas is the highest mountain range in the world"
print(a[5:])    # Output: himalayas is the highest mountain range in the world

Negative Slicing

Negative indices count from the end of the string, with -1 being the last character.

python
x = "The ai is the future"
print(x[-6:-1])  # Output: futur
print(x[-6:])    # Output: future
print(x[:-6])    # Output: The ai is the

Reversing a String

We can reverse a string using slicing with a step of -1.

python
a = "The ai is the future"
print(a[::-1])  # Output: erutuf eht si ia ehT

F-Strings & Format Strings
F-strings are a way to format strings in Python by embedding expressions inside string literals. They are denoted by prefixing the string with the letter f or F. They are the modern, recommended way to format strings.

Basic F-Strings

python
name = "Surendra"
age = 25
print(f"My name is {name} and I am {age} years old.")
# Output: My name is Surendra and I am 25 years old.

The :.2f Format Specifier

The :.2f format specifier formats a floating-point number to a specific number of decimal places. 2f means the number will be formatted to 2 decimal places.

python
a = 3.14159
print(f"The value of pi is approximately {a:.2f}.")
# Output: The value of pi is approximately 3.14.

price = 59
text = f"The price of the item is ${price:.2f}."
print(text)  # Output: The price of the item is $59.00.

Operations Inside F-Strings

We can perform operations and call functions directly inside f-string expressions.

python
x = 10
y = 20
print(f"The sum of {x} and {y} is {x + y}.")
# Output: The sum of 10 and 20 is 30.

name = "python"
print(f"The length of the name is {len(name)}.")
# Output: The length of the name is 6.

print(f"The popular language is now {name.upper()}.")
# Output: The popular language is now PYTHON.

Unpacking a Collection in F-Strings

python
languages = "python"
a, b, c, d, e, f = languages
print(a)  # p
print(b)  # y
print(c)  # t
print(d)  # h
print(e)  # o
print(f)  # n

The .format() Method

The format() method uses placeholders {} in the string and passes values as arguments to fill them.

python
name = "Surendra"
age = 25
print("My name is {} and I am {} years old.".format(name, age))
# Output: My name is Surendra and I am 25 years old.

Old Printf-Style Formatting (Legacy)

This comes from the C language style and is called printf-style formatting. It is not used in modern Python 3.

python
name = "Surendra"
age = 21
print("My name is %s and I am %d years old" % (name, age))
# Output: My name is Surendra and I am 21 years old

# %s  → used for strings
# %d  → used for integers
# %f  → used for floating-point numbers
⚠️Printf-style formatting (%s, %d, %f) is considered legacy. Use f-strings in modern Python 3 code.

String Methods
Strings are immutable in Python — once created, they cannot be changed. However, we can create a new string by modifying the original using various string methods.
upper()
Converts all characters to uppercase
lower()
Converts all characters to lowercase
strip()
Removes leading and trailing whitespace
lstrip()
Removes only leading whitespace
rstrip()
Removes only trailing whitespace
replace()
Replaces all occurrences of a substring
split()
Splits string into a list by delimiter
capitalize()
Capitalizes first character, rest lowercase
casefold()
Aggressive lowercase for caseless matching
count()
Counts occurrences of a substring
find()
Returns lowest index of substring, or -1
endswith()
Returns True if string ends with suffix
startswith()
Returns True if string starts with prefix
join()
Joins a list of strings with a separator
title()
Title-cases each word in the string
isupper() / islower()
Checks if all chars are upper/lower case
isalpha()
True if all characters are alphabetic
isalnum()
True if all characters are alphanumeric
encode()
Returns encoded version of string (bytes)
expandtabs()
Replaces tab characters with spaces

Code Examples

python
a = "Surendra"
print(a.upper())    # Output: SURENDRA
print(a.lower())    # Output: surendra

a = "  surendra  "
print(a.strip())    # Output: surendra
print(a.lstrip())   # Output: surendra   (trailing whitespace remains)
print(a.rstrip())   # Output:   surendra (leading whitespace remains)

a = "The ai is the future"
print(a.replace("ai", "artificial intelligence"))
# Output: The artificial intelligence is the future

print(a.split())
# Output: ['The', 'ai', 'is', 'the', 'future']

a = "the ai is the future"
print(a.capitalize())  # Output: The ai is the future
print(a.title())       # Output: The Ai Is The Future
print(a.casefold())    # Output: the ai is the future

a = "The Then That There"
print(a.count("Th"))   # Output: 4

a = "The ai is the future"
print(a.find("ai"))    # Output: 4
print(a.endswith("future"))   # Output: True
print(a.startswith("The"))    # Output: True

my_list = ["The", "ai", "is", "the", "future"]
print(" ".join(my_list))  # Output: The ai is the future

Modifying Strings (Workaround for Immutability)

Since strings are immutable, we use slicing and concatenation to create a "modified" version.

python
a = "The ai is the future"
b = a[:10] + "amazing " + a[10:]
print(b)  # Output: The ai is amazing the future

String Concatenation

python
a = "The ai is the future"
b = " and it is amazing"
c = a + b
print(c)  # Output: The ai is the future and it is amazing

# Operator precedence: * is evaluated before +
c = a + b * 2
print(c)  # Output: The ai is the future and it is amazing and it is amazing

Escape Strings
Escape strings are used to represent special characters inside a string that would otherwise be difficult or impossible to include. They always start with a backslash \.
\'
Single Quote
\"
Double Quote
\\
Backslash
\n
New Line
\t
Tab
\b
Backspace
python
# Single quote
a = "This is the \' python \' code"
print(a)  # Output: This is the ' python 'code

# Double quote
b = "This is the \" python \" code"
print(b)  # Output: This is the " python "code

# Backslash
c = "This is the \\ python \\code"
print(c)  # Output: This is the \ python \code

# New line
d = "This is the python code \n which is used for the programming"
print(d)
# Output:
# This is the python code
#  which is used for the programming

# Tab
e = "This is the python code \t which is used for the programming"
print(e)  # Output: This is the python code     which is used for the programming

# Backspace
f = "This is the python code \b which is used for the programming"
print(f)  # Output: This is the python cod which is used for the programming

Slicing with Escape Awareness

python
g = "This is the python code which is used for the programming"
print(g[0:10])  # Output: This is th

Operators
Operators are used to perform operations on variables and values. Python has seven categories of operators, each serving a different purpose.

Arithmetic Operators

OperatorNameExampleOutput
+Addition10 + 515
-Subtraction10 - 55
*Multiplication10 * 550
/Division10 / 52.0
%Modulus10 % 50
**Exponentiation10 ** 5100000
//Floor Division10 // 52

Comparison Operators

python
x = 10
y = 20
print(x == y)  # False — equal to
print(x != y)  # True  — not equal to
print(x > y)   # False — greater than
print(x < y)   # True  — less than
print(x >= y)  # False — greater than or equal to
print(x <= y)  # True  — less than or equal to

Logical Operators

python
a = True
b = False
print(a and b)  # False — both must be True
print(a or b)   # True  — at least one must be True
print(not a)    # False — opposite of a

Assignment Operators

python
x = 10
x += 5   # x = 15  (add and assign)
x -= 3   # x = 12  (subtract and assign)
x *= 2   # x = 24  (multiply and assign)
x /= 4   # x = 6.0 (divide and assign)
x %= 3   # x = 0.0 (modulus and assign)
x **= 2  # x = 0.0 (power and assign)
x //= 2  # x = 0.0 (floor divide and assign)

Bitwise Operators

python
a = 5   # binary: 0101
b = 3   # binary: 0011
print(a & b)   # Output: 1  (AND  → 0001)
print(a | b)   # Output: 7  (OR   → 0111)
print(a ^ b)   # Output: 6  (XOR  → 0110)
print(~a)      # Output: -6 (NOT  → 11111010)
print(a << 1)  # Output: 10 (Left shift  → 1010)
print(a >> 1)  # Output: 2  (Right shift → 0010)

Identity Operators

Identity operators compare the memory location of two objects, not their values.

python
x = 10
y = 10
print(x is y)      # True  — same memory object (small ints are cached)
print(x is not y)  # False

Membership Operators

Membership operators test if a value is present in a sequence (list, tuple, string, etc.).

python
my_list = [1, 2, 3, 4, 5]
print(3 in my_list)      # True
print(6 in my_list)      # False
print(3 not in my_list)  # False
print(6 not in my_list)  # True

# Also works with strings
my_string = "Hello, World!"
print("Hello" in my_string)      # True
print("Python" in my_string)     # False
print("Hello" not in my_string)  # False
print("Python" not in my_string) # True

🎯 Day 1 Mini Project
User Information Card
This is the assignment of the mini project — implementing everything I learned on Day 1 of Python. It uses operators, escape strings, and format strings to create a simple formatted card that includes the name, age, city, and hobby of the user. The card is decorated with escape strings and operators.
python
# Taking the user input for the card details
name  = input("Enter your name: ")
age   = int(input("Enter your age: "))
city  = input("Enter your city: ")
hobby = input("Enter your hobby: ")

# Printing the formatted card using f-strings and escape strings
print("\n" + "=" * 30)       # line of 30 = signs to decorate the card
print("     user information card")
print("=" * 30)
print(f"Name  : {name}")     # name of the user
print(f"Age   : {age}")      # age of the user
print(f"City  : {city}")     # city of the user
print(f"Hobby : {hobby}")    # hobby of the user
print("=" * 30)
